Settings

Page id: S120

In this page, the settings of the application can be set.

The following settings are available:

  • Url
  • Username
  • Password
  • Demo mode

Url

This is the url of the webservice.
Example: https://company.com:8080/web/service.dll

Username

The username if basic authentication of the webservice is used.

Password

The password if basic authentication of the webservice is used.

Demo mode

Use this if you want to demo the application without the use of a backend.

To confirm the settings, press save.
